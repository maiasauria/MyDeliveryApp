# MyDeliveryApp

Aplicacion desarrollada en Kotlin para Android, utilizando MVVM, Hilt, Room y Retrofit.

## Features

- Listado de productos
- Detalles de productos
- Carrito de compras
- Checkout
- Registro de usuarios
- Historial de pedidos

## Tech Stack

- Kotlin
- Hilt
- Retrofit & Gson
- Glide
- Material Components
